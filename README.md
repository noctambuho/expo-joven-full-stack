# Expo Joven Full Stack 2026 - Proyecto de práctica

## Configuración del entorno de desarrollo

- IDE: VS-Codium (Fedora 43)
- Extensiones:
    - `anseki.vscode-color`: Selector de color avanzado.
    - `ecmel.vscode-html-css`: Soporte de clases CSS para HTML.
    - `google.geminicodeassist`: Asistente de codificación con IA de Google. Solo para consultas rápidas y ayudas automáticas en la compleción de código.
    - `gydunhn.html-essentials`: Snippets y herramientas esenciales para HTML.
    - `html-validate.vscode-html-validate`: Validador de código HTML.
    - `naumovs.color-highlight`: Resalta los colores en el código.
    - `pranaygp.vscode-css-peek`: Permite ver definiciones de CSS desde el HTML.
    - `stylelint.vscode-stylelint`: Linter para CSS, SCSS y Less.
    - `vincaslt.highlight-matching-tag`: Resalta la etiqueta de apertura/cierre correspondiente.
    - `yandeu.five-server`: Servidor de desarrollo con recarga en vivo.

## Tecnologías utilizadas

Este es un proyecto de práctica enfocado en los fundamentos del desarrollo web, utilizando:

- **HTML5**
- **CSS3**
